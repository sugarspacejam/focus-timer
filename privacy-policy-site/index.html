Done in 5

Privacy Policy

Done in 5 is a local focus timer app.

Data Collection

Done in 5 does not require account creation.

Done in 5 stores app state locally on your device, including:

  • task names you create
  • timer state
  • camera accountability setting
  • custom away voice lines
  • local usage stats related to completed and failed focus contracts

Camera Use

If you enable camera accountability, the app uses the front camera while the app is open and an active focus contract is running.

Camera processing is used only to determine whether you remain in frame during the active contract.

Camera accountability is foreground-only in the current app behavior.

Notifications

Done in 5 requests notification permission so it can notify you when a focus contract completes.

Purchases

Done in 5 is intended to be sold as a paid download. The current shipped app path does not require account-based purchases to use the app.

Third Parties

Done in 5 uses Apple system frameworks such as notifications, camera, and speech synthesis.
